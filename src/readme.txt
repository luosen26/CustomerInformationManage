                                                                           通讯录管理系统
	该系统有如下功能：
		1.登录功能
		2.注册用户功能
		3.找回密码功能
		4搜索功能：
		1）搜索框里什么都不输入，则搜索全部
		2）搜索框里输入数字，则按联系人编号搜索
		3）搜索框里输入字符串，则按联系人名字搜索
		5.登录以后自动显示全部联系人
		6.动态显示用户名
		7.修改用户信息
		8.修改用户密码
		9.添加联系人
		10.修改联系人信息
		11.删除联系人
		12.退出通讯录管理系统
			        	         Contacts Management System
	The system has the following functions:
		1.Login function
		2. Registered User Function
		3. Retrieve Password Function
		4Search function:
		1) Enter nothing in the search box, then search all
		2) Enter numbers in the search box, then search by contact number
		3) Enter a string in the search box, then search by contact name
		5. Automatically display all contacts after login
		6.Dynamic display user name
		7. Modify user information
		8. Modify user password
		9. Add Contact
		10. Modify contact information
		11. delete contact
		12. Exit the address book management system
